### Icon基本用法

```tsx
<Icon icon={"coffee"}/>
```

### Icon 属性

| 属性  | 说明 |  类型  |                            可选值                            | 默认值 |
| :---: | :--: | :----: | :----------------------------------------------------------: | :----: |
| theme | 颜色 | string | "primary" \| "secondary" \| "success" \| "info" \| "warning" \| "danger" \| "light" \| "dark" |   —    |
